快速安装

terminal
# 使用 Ollama 拉取模型
ollama pull xiaobai/unclec-stone-2.0-114514b

# 或者使用 CLI 直接运行
ollama run xiaobai/unclec-stone-2.0-114514b

# API 调用示例

为什么选择 UncleC-stone-2.0-114514B

卓越中文能力

针对中文语境深度优化，在中文理解、生成和推理任务上表现优异，支持成语、谚语和文化背景理解

代码专家
