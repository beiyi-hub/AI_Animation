Your own personal AI assistant. Any OS. Any Platform. The lobster way.

OpenClaw is a personal AI assistant you run on your own devices. It answers you on the channels you already use. It can speak and listen on macOS/iOS/Android, and can render a live Canvas you control.

Highlights

Local-first Gateway

Single control plane for sessions, channels, tools, and events. Your data stays on your device.

Multi-channel Inbox

WhatsApp, Telegram, Slack, Discord, Signal, iMessage, IRC, Teams, Matrix, WeChat, QQ, and more.

Multi-agent Routing

Route inbound channels/accounts/peers to isolated agents with workspace and per-agent sessions.

Voice Wake + Talk Mode

Wake words on macOS/iOS and continuous voice on Android with ElevenLabs + system TTS fallback.

Live Canvas

Agent-driven visual workspace with A2UI. See and control your AI assistant's thinking visually.

First-class Tools

Browser, canvas, nodes, cron, sessions, and Discord/Slack actions built-in.

Companion Apps

macOS menu bar app + iOS/Android nodes for seamless mobile integration.

Onboarding + Skills

Onboarding-driven setup with bundled/managed/workspace skills via ClawHub.

Security Model

Important: Default behavior on Telegram/WhatsApp/Signal/iMessage/Microsoft Teams/Discord/Google Chat/Slack:

  • DM pairing: Unknown senders receive a short pairing code and the bot does not process their message.
  • Approve with: openclaw pairing approve <channel> <code>
  • Public DMs: Require explicit opt-in with dmPolicy="open"

Run openclaw doctor to surface risky/misconfigured DM policies.

Security: Before exposing anything remotely, read the full Security, Sandboxing, and Configuration guides.

Models

While many providers and models are supported, prefer a current flagship model from the provider you trust and already use.

Supported providers include: OpenAI (ChatGPT/Codex), Anthropic, Google, Mistral, and 200+ models via OpenRouter.
